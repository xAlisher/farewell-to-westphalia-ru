# Farewell to Westphalia (RU)

This repository contains a Russian translation and adaptation of
*Farewell to Westphalia* by Jarrad Hope and Peter Ludlow.

The purpose of this project is to make the ideas of post-nation-state
governance and crypto sovereignty accessible to Russian-speaking readers,
using clear, precise, and readable language.

This is not a literal, word-for-word translation.
The text is adapted with priority given to clarity, rhythm, and conceptual
accuracy, while preserving the authors’ original intent and arguments.

## About the book

*Farewell to Westphalia* examines the historical foundations of the
nation-state system and explores how new technologies, particularly
cryptographic and decentralized systems, enable alternative forms of
governance beyond the Westphalian model.

The book approaches governance as a human-made technology: one that can
be analyzed, stress-tested, and redesigned.

## Structure

- original/ — source text split into chapters
- ru/ — Russian text split into chapters
- docs/ — glossary, style rules, and translation decisions

A shared glossary is maintained to ensure consistent use of key terms
throughout the text.
